### 깃 구성요소4

1. 원격 저장소 정보 등록(url)
2. git remote add remote별명 remote주소 <- 주로 사용하는 remote별명은 origin이름으로 사용한다.
3. 그 후에 저장된 주소로 push 해준다.
4. git push -u origin master
5. 그 후에 실제 repository에 올라갔는지 깃허브에서 직접 확인.
